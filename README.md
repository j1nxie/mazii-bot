# mazii-bot

a Discord bot for searching Japanese words, using [mazii](https://mazii.net/).

## usage

### prerequisites

```sh
pnpm install
```

### development

this example can be run with `tsc-watch` to watch the files and automatically restart your bot.

```sh
pnpm run watch:start
```
